# Codeforces 用户信息查询工具 (Codeforces Rating Clawer)

本项目是一个基于 C 语言编写的 Codeforces 用户数据爬取与查询工具。它可以抓取用户的个人基本信息、比赛记录和做题记录，并生成HTML报告。
* 本项目为湘潭大学计算机学院2025级计算机科学与技术1班陈羽C语言程序设计大作业。

## ✨ 主要功能

- **多用户并发查询**：支持同时查询最多 100 名 Codeforces 用户（用户名不区分大小写，空格分隔）。
- **生成概览列表**：查询多名用户，生成 `index.html` 主页，概览各个用户的当前等级分（Rating）、最高等级分、头衔及参赛次数等信息。
- **个人详细数据报告**：为每位用户生成专属的 `${用户名}_report.html` 交互数据页面：
  - **基本信息统计**：当前等级分、最高等级分、近 180 天参赛数据、通过题目总数等。
  - **等级分变化折线图**：动态展示加入 Codeforces 以来的历次比赛等级分变化轨迹。
  - **过题难度分布柱状图**：按“全部”、“最近一年”、“最近 180 天”、“最近 1 个月”等时间维度切换，展示已解决题目的难度分布。
  - **详细比赛与补题记录**：列出该用户参加过的每一场比赛状态，并标记每道题的过题情况：
    - 🟢 绿色背景（OK）：赛内通过
    - 🟡 黄色背景（After）：赛后补题通过
    - ⚪ 灰色背景（No）：未通过
    
## 🛠️ 技术栈与依赖
- **开发语言**: C语言
- **构建工具**: CMake
- **网络通信框架**: `libcurl`
- **JSON 解析**: `cJSON` 
- **图表渲染库**: `ECharts` 

## 🚀 编译与使用

建议使用 CLion 或者拥有 MinGW 编译环境的 Windows 系统进行编译。
项目中调用了 `Windows.h` 获取控制台彩色文字。

### 编译步骤

可以使用 CLion 自动重新加载 `CMakeLists.txt` 构建。
注意正确配置 `libcurl` 的 `include` 和 `lib` 路径到项目中。

### 运行方式

**交互式运行**  
直接双击运行 `./cmake-build-release/Codeforces用户查询.exe`。
程序内置彩色的控制台引导排版，只需输入想要查询的 Codeforces 用户名并回车，多个用户使用空格隔开（例如输入：`Benq cry tourist`）。


## 📂 生成文件结构输出

运行时会在执行目录下自动创建一系列 HTML报告：
- `index.html` : 全部查询用户的列表页。
- `<用户名>_report.html` : 包含数据分析及图表展示的独立报告页面。
